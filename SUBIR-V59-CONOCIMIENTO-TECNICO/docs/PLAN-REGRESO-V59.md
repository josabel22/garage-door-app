# Plan de regreso V59

Si algo falla despues de subir:

1. No tocar la app principal.
2. Eliminar o renombrar solamente la carpeta publicada `/conocimiento`.
3. Limpiar en el navegador solo las claves del modulo si fuera necesario:
   - `mg_knowledge_v59`
   - `mg_knowledge_source_v59`
   - `mg_knowledge_session_v59`
4. Verificar que `/index.html` sigue funcionando.
5. Restaurar desde el respaldo solo si hubo cambios fuera de la carpeta `conocimiento`.

Esta version esta disenada para que el regreso sea borrar la carpeta del modulo, no restaurar toda la aplicacion.
